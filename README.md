## Dependencies
[bazel](https://bazel.build/)

Follow the install instruction. I use openSUSE so:
```
sudo zypper install bazel
```

[cpplint](https://github.com/cpplint/cpplint)

Install with
```
pip install cpplint
```

## Lint

cpplint --recursive --filter=-legal ./
